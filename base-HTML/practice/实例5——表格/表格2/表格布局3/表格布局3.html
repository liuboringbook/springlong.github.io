我“和平使命-2007”军演部队启程 徐才厚送行

发表时间：2007-07-20 21:57

    新华社新疆吐鲁番7月20日电 （记者王建民、刘昕）人民解放军参加“和平使命―2007”上海合作组织成员国武装力量联合反恐军事演习部队首批铁路输送的300多名官兵，20日在新疆吐鲁番启程开赴俄罗斯境内演习地域。中共中央书记处书记、中央军委副主席徐才厚出席送行仪式，代表胡锦涛主席和中央军委，向参加演习的全体指战员表示亲切慰问，要求参演官兵以高昂的斗志、良好的素质、过硬的作风和优异的成绩，为祖国争光，为军旗添彩。

    这次演习，是人民解放军历史上首次成建制较大规模、远距离、多军兵种出境参加联合军演，也是上海合作组织全体成员国参加的首次联合反恐军事演习。 　　

    上午11时，在雄壮的国歌声中，送行仪式在吐鲁番火车站隆重举行。车站广场上，彩旗猎猎，军歌嘹亮，全体参演官兵军容严整、士气高昂。满载装备物资的军列停靠在站台上。在送行仪式上，陆军战斗群代表和空军战斗群代表先后发言，表示一定牢记党和人民重托，坚决完成演习任务。新疆维吾尔自治区领导代表自治区党委、政府和各族人民欢送部队。11时30分，徐才厚发布出发命令，在当地各族人民群众的欢送下，参演将士精神抖擞地登车启程，从天山脚下踏上万里征途。 　　

    徐才厚在送行仪式上指出，派出精锐部队参加这次联合反恐军演，是党中央、胡主席作出的重要决策，也是我军一项重大军事行动，参演部队任务艰巨、使命光荣。胡主席对我军参加这次联合反恐军演高度重视，作出重要指示，为我们完成演习任务指明了方向，是对全体参演指战员发出的最有力的动员令。我们要切实用胡主席重要指示统一官兵思想和行动，高标准做好演习各项工作。
