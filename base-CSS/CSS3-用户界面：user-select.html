CSS3用户界面：user-select

该属性用于设置用户是否能够选中文本。

浏览器支持情况：

(-ms-) IE10+（2012/09/04）

(-mz-) Firefox2+（2006/10/24）

Chrome 54+（2016/10/12） (-webkit-) Chrome4+（2010/01/25）

(-webkit-) Safari3.1+（2008/03/18）

Opera 41+（2016/10/24） (-webkit-) Opera15+（2013/07/02）

(-webkit-) IOS3.2+（2010/04/03）

Android 5+（2017/01/01） (-webkit-) Android2.1+（2009/10/26）

user-select:
	text (默认值，可以选择文本) |
	none (文本不能被选择) |
	all (所有内容作为一个整体时可以被选择。如果双击或者在上下文上点击子元素，那么被选择的部分将是以该子元素向上回溯的最高祖先元素) |
	element (可以选择文本，但选择范围受边界的约束);

IE6-9不支持该属性，但支持使用标签属性 onselectstart="return false;" 来达到 user-select:none; 的效果；

除Chrome和Safari外，在其它浏览器中，如果将文本设置为 user-select:none;，则用户将无法在该文本块中开始选择文本。不过，如果用户在页面的其他区域开始选择文本，则用户仍然可以继续选择将文本设置为 user-select:none; 的区域文本；目前较新版本的Firefox浏览器已经修复该问题。
