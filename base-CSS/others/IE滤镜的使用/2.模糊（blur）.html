CSS应用-滤镜的使用（模糊）

模糊属性blur――相关参数说明
filter:blur(add=?,direction=?,strength=?)
参数 说明
add 参数值为true和false，指定图片是否显示原始图片，true为显示，false为不显示，默认设置为true！
direction 设置模糊的方向，模糊效果是按照顺时针方向进行的。其中0。表示垂直向上；每45。为一个单位，默认值是向左270。
strength 设置模糊强度，只能用整数来指定，代表有多少像素的宽度将受到模糊影响，默认值为5像素！

（模糊设置）blur属性――add：

正常情况下！

filter:blur(add=true);
显示原图片！

filter:blur(add=false);
不显示原图片！

（模糊设置）blur属性――direction：

正常情况下！

filter:blur(direction=0);
模糊方向为0。

filter:blur(direction=45);
模糊方向为45。

filter:blur(direction=90);
模糊方向为90。

filter:blur(direction=135);
模糊方向为135。

filter:blur(direction=180);
模糊方向为180。

filter:blur(direction=225);
模糊方向为225。

filter:blur(direction=270);
模糊方向为270。

filter:blur(direction=315);
模糊方向为315。

filter:blur(direction=360);
模糊方向为360。

（模糊设置）blur属性――strength：

正常情况下！

filter:blur(strength=5);
模糊强度为5像素

filter:blur(strength=10);
模糊强度为10像素

filter:blur(strength=30);
模糊强度为30像素

filter:blur(strength=50);
模糊强度为50像素
