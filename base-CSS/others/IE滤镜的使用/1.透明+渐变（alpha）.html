CSS应用-滤镜的使用（透明+渐变）

透明度/渐变属性Alpha――相关参数说明
filter:alpha(opacity=?,finishopacity=?,style=?,startx=?,starty=?,finishx=?,finishy=?)
参数 说明
opacity 设置图片或文字的不透明度，参数值从0到100，0代表完全透明，100代表完全不透明！
finishopacity 该参数用来说明透明渐变结束时的不透明度！参数值也从0到100！
style 指定透明渐变的形状特征：0――统一形状、1――线形、2――放射状、3――长方形
startx 渐变透明效果的开始X坐标
starty 渐变透明效果的开始Y坐标
finishx 渐变透明效果的结束X坐标
finishy 渐变透明效果的结束Y坐标

（不透明度设置）Alpha属性――opacity：

正常情况下！

filter:alpha(opacity=100)
100%不透明

filter:alpha(opacity=80)
80%不透明

filter:alpha(opacity=60)
60%不透明

filter:alpha(opacity=30)
30%不透明

filter:alpha(opacity=10)
10%不透明

filter:alpha(opacity=0)
0%不透明

（透明形状）Alpha属性――style：

正常情况下！

filter:alpha(opacity=60);
不透明度60% 默认形状

filter:alpha(opacity=60,style=0)
不透明度60% 统一形状

filter:alpha(opacity=60,style=1)
不透明度60% 线形

filter:alpha(opacity=60,style=2)
不透明度60% 放射状

filter:alpha(opacity=60,style=3)
不透明度60% 长方形

（终止不透明度设置）Alpha属性――finishopacity：

正常情况下！

filter:alpha(opacity=60,finishopacity=100,style=1);
不透明度60% 终止不透明度100% 线形渐变

filter:alpha(opacity=60,finishopacity=90,style=1);
不透明度60% 终止不透明度90% 线形渐变

filter:alpha(opacity=60,finishopacity=60,style=1);
不透明度60% 终止不透明度60% 线形渐变

filter:alpha(opacity=60,finishopacity=30,style=1);
不透明度60% 终止不透明度30% 线形渐变

filter:alpha(opacity=60,finishopacity=10,style=1);
不透明度60% 终止不透明度10% 线形渐变

filter:alpha(opacity=60,finishopacity=0,style=1);
不透明度60% 终止不透明度0% 线形渐变
