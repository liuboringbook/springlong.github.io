CSS应用-滤镜的使用（翻转+灰度+反转+X射线）

翻转+灰度+反转+X射线
filter:fliph flipv gray invert xray;
属性 说明
fliph 水平翻转
flipv 垂直翻转
gray 灰度模式：将图片的颜色去除，产生黑白的效果！
invert 反转：将对象颜色的饱和度和亮度值设置为反转，产生类似于底片的效果！
xray X射线效果

（水平翻转）fliph：

正常情况下！

filter:fliph;
水平翻转！

（垂直翻转）flipv：

正常情况下！

filter:flipv;
垂直翻转！

（灰度模式）gray：

正常情况下！

filter:gray;
灰度模式

（反转）invert：

正常情况下！

filter:invert;
反转！

（X射线）xray：

正常情况下！

filter:xray;
X射线效果！
