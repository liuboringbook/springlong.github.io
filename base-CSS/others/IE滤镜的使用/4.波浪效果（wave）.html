CSS应用-滤镜的使用（波浪效果）

波浪效果
filter:wave(add=?,freq=?,lightstrength=?,phase=?,strength=?);
属性 说明
add 是否显示源图像!true表示显示源图像；false表示不显示源图像；默认值为false，不显示源图像！
freq 设置产生的波浪的条目，数值越多，波浪的条目就越多！默认值为3，表示有三条波浪！
lightstrength 设置波浪的光照强度，参数值从0到100！0表示光照最弱，波浪不显示颜色；100表示光照最强，波浪显示纯黑色！默认值为100，呈纯黑显示！
phase 设置正弦波开始的偏移量，取值从0到100！
strength 设置振幅，也称波浪的强度！值从0到100！默认值为0，波浪呈水平显示！值越大，波浪的强度也就越大！

（波浪效果）wave属性――add：

正常情况下！

filter:wave(add=true);
显示源图像！

filter:wave(add=false);
不显示源图像！

（波浪效果）wave属性――freq：

正常情况下！

filter:wave(freq=3);
3条波浪线！

filter:wavefreq=6);
6条波浪线！

（波浪效果）wave属性――lightstrength：

正常情况下！

filter:wave(lightstrength=0);
光照强度为0！

filter:wave(lightstrength=30);
光照强度为30！

filter:wave(lightstrength=60);
光照强度为60！

filter:wave(lightstrength=80);
光照强度为80！

filter:wave(lightstrength=100);
光照强度为100！

（波浪效果）wave属性――phase：

正常情况下！

filter:wave(strength=70);
波浪的振幅为：70！

filter:wave(phase=0);
正弦波的起始位置：360。*0% = 0。

filter:wave(phase=12.5);
正弦波的起始位置：360。*12.5% = 45。

filter:wave(phase=25);
正弦波的起始位置：360。*25% =90。

filter:wave(phase=37.5);
正弦波的起始位置：360。*37.5% = 135。

filter:wave(phase=50);
正弦波的起始位置：360。*50% = 180。

filter:wave(phase=62.5);
正弦波的起始位置：360。*62.5% = 225。

filter:wave(phase=75);
正弦波的起始位置：360。*75% = 270。

filter:wave(phase=87.5);
正弦波的起始位置：360。*87.5% = 315。

filter:wave(phase100);
正弦波的起始位置：360。*100% = 360。
