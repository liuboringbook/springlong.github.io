CSS应用-滤镜的使用（遮罩+透明）

遮罩+透明

filter:mask(color=?);

filter:chroma(color=?);
属性 说明
color 设置遮障或透明的颜色！
  遮障：在一个区块中应用了遮障效果，则该区块将被指定的颜色所覆盖，被覆盖的元素（包括文字和图片）都将呈透明显示，即页面的背景颜色是什么，则他们就显示为什么颜色！
  透明：在一个区块中应用了透明效果，则该区块中与该颜色一样的元素都将呈透明显示（注：边缘部分显示为原本的颜色），即页面的背景颜色是什么，则他们就显示为什么颜色！
  注意：边缘光晕必须针对某一区块中的文本或图片，即要设置一个区块的width或height属性，才能正确显示出阴影的效果！
正常情况下：

这是正常情况下的显示效果！

0.默认设置下：filter:mask();

这是遮罩的显示效果！

1.遮罩颜色为灰色：filter:mask(color=gray);

这是遮罩的显示效果！

2.遮罩颜色为蓝色：filter:mask(color=blue);

这是遮罩的显示效果！

3.遮罩颜色为橙色：filter:mask(color=orange);

这是遮罩的显示效果！

4.遮罩颜色为白色：filter:mask(color=white);

这是遮罩的显示效果！

0.默认设置下：filter:chroma();

这是透明的显示效果！

1.透明颜色为黑色：filter:chroma(color=black);

这是透明的显示效果！

2.透明颜色为灰色：filter:chroma(color=gray);

这是透明的显示效果！
