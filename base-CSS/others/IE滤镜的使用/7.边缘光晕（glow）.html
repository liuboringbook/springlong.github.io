CSS应用-滤镜的使用（边缘光晕）

边缘光晕
filter:glow(color=?,strength=?)
属性 说明
color 设置边缘光晕的颜色，默认值为红色！
strength 设置光晕的长度（单位：像素），可设置的值有0~255！
  注意：边缘光晕必须针对某一区块中的文本或图片，即要设置一个区块的width或height属性，才能正确显示出阴影的效果！

这是边缘光晕的显示效果！
