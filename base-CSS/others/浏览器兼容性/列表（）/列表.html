浏览器兼容——列表

问题一： 在所有浏览器中，无论是有序列表还是无序列表，它们的ol或ul都被浏览器默认设置了26px左右的左填充（即：padding-left:26px;），从而达到显示列表符号的目的。要想让列表正常显示出列表符号，还必须满足以下几个条件：

  1. 必须确保li元素的overflow属性值为visible，如果设置其为hidden，那么列表符号将被作为多余文本而被隐藏掉；
  2. 必须确保li元素的list-style-type属性不为none，这是显示列表符号的必要条件；
  3. 在IE6/7浏览器下，必须保证li元素的float属性为none，否则列表符号将无法显示；
  4. 在IE6/7浏览器下，必须保证li元素的height属性值为auto，否则对于有序列表符号而言，将始终仅显示序号值1或i，具体所显示的序号值视li元素的list-style-type属性值而定！
