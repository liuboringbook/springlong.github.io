面板标题

我们可以通过以下两种方式来添加面板标题：

  • 使用 .panel-heading class 可以很简单地向面板添加标题容器。to easily add a heading container to your panel.
  • 使用带有 .panel-title class 的 <h1>-<h6> 来添加预定义样式的标题。
