jQuery代码示例：.each()

.each(function(){index, ele})

遍历元素集合，为每个元素执行一次回调函数，并返回原jQuery对象。（返回类型：jQuery）

参数function(){index, ele}：（类型：Function）回调函数；

//下面的代码将针对box容器下的所有后代元素中，索引值为偶数（0、2、4...）的段落p的背景颜色设置为紫色
$("#box p").each(function(index, ele)
{
    if(index % 2 === 0)
    {
        $(ele).css("background-color", "purple");
    }
});


//下面的代码将针对box容器下的所有后代元素中，索引值为奇数（1、3、5...）的段落p的背景颜色设置为绿色
$("#box p").each(function(index, ele)
{
    if(index % 2 !== 0)
    {
        $(this).css("background-color", "green");
    }
});


//下面的代码将针对box容器下的所有后代元素中，索引值为偶数（0、2、4...）的段落p的背景颜色设置为橙色
$("#box p").filter(":even").css("background-color", "orange");
