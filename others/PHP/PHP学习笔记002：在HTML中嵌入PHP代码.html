PHP学习笔记002：在HTML中嵌入PHP代码

在一个php文件中，如果没有嵌入后端代码，那么我们完全可以理解为——它就是一个html文件，只是增加了可以书写后端代码的能力。

如果需要在php文件中执行后端代码，我们可以通过下面三种方式来实现：

  1. 表达式语法 
    <?=表达式 ?>

    该种方式将在代码位置处输出表达式的计算结果，同时也支持变量、常量、方法等的直接输出。

    例如，下面的语句代码将直接输出为6：

    <?=3 + 3 ?>
  2. 语句语法 
    <?php 语句 ?>

    该种方式将在代码位置处输出目标语句所执行的结果。

    例如，下面的语句代码将输出1~10这10个数字使用单个空格隔开的结果：

    <?php
    for($i = 1; $i <= 10; $i++)
    {
        echo $i." ";
    }
?>
    注意：

    在支持简写的服务器上，我们可以将上面的<?php ?>代码简写为<? ?>。不过为了达到最好的兼容性，我们最好使用标准形式，而不是简写形式。

  3. 组合语法

    该种方式就是将表达式语法、语句语法、HTML代码组合在一起进行书写。

    例如，下面的语句代码将输出10个段落，且段落内容分别为1~10：

    <?php
    for($i = 1; $i <= 10; $i++)
    {
?>
<p><?=$i ?></p>
<?php
    }
?>
